# Design System

## Theme
Light mode default. The environment is bright and premium, highlighting the colorful clothing items.

## Color Strategy
Restrained with vibrant accents. The primary color is a bright, energetic hue against tinted neutral backgrounds (off-white, very subtle cool gray). Using OKLCH color space.

## Typography
Clean, modern aesthetic. Generous line height and distinct hierarchy.

## Layout
Spacious grids, clear visual hierarchy, subtle borders rather than heavy cards.

## Components
- Buttons: Solid vibrant primary, subtle outlined secondary.
- Cards: Clean edges, subtle hover lift, no heavy shadows.
- Forms: Clear inputs with prominent focus states.
